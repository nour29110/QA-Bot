# QA Bot That Leverages the LangChain and LLM to Answer Questions from Loaded Documents

## Various components were used, including document loaders, text splitters, embedding models, vector databases, retrievers, and Gradio as the front-end interface, to build a fully functional QA system. By leveraging LangChain and an LLM, the bot reads and understand the content of loaded PDF documents, enabling it to provide precise answers to user queries.
